# Hubblepy - Python wrapper for the Hubblesite.org API

`hubblepy` is a straightforward and easy-to-use API wrapper for the [Hubblesite](http://hubblesite.org/) API.

## Requirements

* Python 3.4+
* `requests>=2.18`
